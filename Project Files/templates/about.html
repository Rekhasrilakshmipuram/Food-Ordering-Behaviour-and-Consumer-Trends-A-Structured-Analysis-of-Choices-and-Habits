About the Project

Behind FoodPulse Analytics

A deep dive into food delivery data — uncovering patterns, trends, and actionable insights.

🎯 Project Overview

FoodPulse Analytics is a comprehensive food delivery data analysis project. We analyzed order data from a major food delivery platform to extract insights about delivery performance, customer preferences, restaurant metrics, and operational efficiency.

The project combines interactive Tableau dashboards with a modern Flask web application, making data exploration intuitive and visually engaging.

📊 Key Metrics Analyzed

  • Order volume & revenue trends over time
  • Average delivery time by restaurant & cuisine
  • Customer satisfaction ratings distribution
  • Peak ordering hours & day-of-week patterns
  • Cuisine type popularity & revenue share
  • Restaurant performance benchmarking

⚙️ Technology Stack

  • Visualization: Tableau Public
  • Backend: Python Flask
  • Frontend: HTML5, CSS3, JavaScript
  • Embedding: Tableau Embedding API v3
  • Data Source: FoodHub Order Dataset
  • Design: Glassmorphism, Dark Theme

📁 Data Source

The analysis is based on the FoodHub order dataset containing delivery records with attributes including restaurant name, cuisine type, order cost, delivery time, customer rating, and order timestamps.
